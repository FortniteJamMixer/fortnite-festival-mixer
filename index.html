1. How the Camelot Wheel Works

Camelot is a DJ key system: each key becomes a number 1‚Äì12 and a letter (A = minor, B = major). The closer two codes are on the wheel, the smoother they mix.

  • Tier 1 ‚Äì Perfect: Same number & same letter (e.g. 8A ‚Üí 8A).
  • Tier 2 ‚Äì Near: Same letter, ¬±1 number (e.g. 8A ‚Üí 7A or 8A ‚Üí 9A).
  • Tier 3 ‚Äì Swap: Same number, different letter (e.g. 8A ‚Üí 8B).

When you pick a main track, we rate every other track by key distance and BPM distance. The Camelot Tier toggle decides how strict we are:

  • Tier 1: Same number & letter only.
  • Tiers 1‚Äì2: Tier 1 + neighbors with same letter.
  • All Tiers: Adds Tier 3 and creative keys.

2. Priority: Tempo vs Camelot

Use the Priority button to switch between:

  • Tempo: Best overall matches first (key + BPM), tie-broken by BPM distance.
  • Camelot: Best tiers first (1 ‚Üí 2 ‚Üí 3), then overall score, then BPM distance.
